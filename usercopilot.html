UserCopilot

Project information

  • Industory: Virtual Assistant
  • Client: USA
  • Services: Web Development, AI development, Chatbot Development.
  • Frameworks: Langchain, Python(Django, Flask), AWS(EC-2), ReactJS, AWS Lambda, AWS S3 bucket

Description

Businessman who need a virtual assistant for their website can purchase a virtual agent trained on their custom data, available at the spot to introduce GenAI to innovate your business. From data scrapping to chatbot deployment, every thing is automated.

Features and Functionalities

1. Password less Login/Signup

Access your dashboard using magic link sent to your email account.
We used services of Twilio & SendGrid for this integration.

Benefits

  • Seamless User experience
  • Enhance Security
  • Boosted productivity
  • Multifactor Authentication

2. Data Scraping

Take the child URLs, scrape their data, and store them on the vectorDB for semantic search or similarity index.

Key Features

  • Knowledge Base Extraction
  • Semantic Search
  • Pinecone Architecture
  • Metadata
  • Vector Index Manipulation

3. Custom Agent Setup

Setting up your custom Virtual Agent according to your business requirements. Most advanced and amazing features

Key Features

  • Personalize your company profile
  • Select Avatar and store it in AWS S3 box
  • Zapier Help Desk Integration
  • Sub Users Permissions
  • Human Hand-off Feature
  • Appointment Forum

4. ChatGPT Settings

Set you agent ChatGPT settings by prompting and Top-K.

Key Features

  • System Prompt to instruct ChatGPT how to answer website visitor question.
  • User Prompt to provide additional instructions to your agent.
  • GPT Model 3.5 Turbo
  • Temperature Setting (Restricted, Conservative, Creative)
  • Top-K Settings

5. Customize Automated Chatbot

Customize Chatbots by setting welcome messages, themes, styles, and brand references.

Key Features

  • Default Message Bubbles
  • Place holder text
  • Bot Icon style
  • Enable custom features
  • Automated hex colors tools
  • Human Hand-off form

6. Deploy UserCopilot

In the end, deploy your custom built Chatbot.

Key Features

  • We had made a deployment script which you can copy in your website header to deploy the chatbot into your web.
  • JS Script Deployment
  • Download Wordpress Plugin to attach to your website
  • Encrypted Key for security & spam protection
